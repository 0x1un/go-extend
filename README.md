# go-extend

go语言扩展包，收集一些常用的操作函数，辅助更快的完成开发工作，并减少重复代码。

规范:
	
- 包名统一使用 `ex` 前缀， 避免与官方包冲突。
- 包目录下 `doc.go` 作为包说明文档。
